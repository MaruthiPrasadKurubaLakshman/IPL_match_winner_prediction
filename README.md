# IPL Match Winner Prediction

## Objective
The objective of this project is to predict the winning probability of an IPL team during a live match using Machine Learning techniques based on match conditions such as runs left, balls left, wickets remaining, current run rate, and required run rate.

---

## Technologies Used
- Python
- Jupyter Notebook
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Pickle

---

## Machine Learning Algorithm Used
This project uses classification algorithms to predict match-winning probabilities.

Examples:
- Logistic Regression
- Random Forest Classifier

---

## Features Used
The prediction model uses the following match details:

- Batting Team
- Bowling Team
- Host City
- Target Score
- Current Score
- Overs Completed
- Wickets Fallen
- Runs Left
- Balls Left
- Current Run Rate (CRR)
- Required Run Rate (RRR)

---

## Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Model Training
5. Probability Prediction
6. Streamlit Web App Deployment
